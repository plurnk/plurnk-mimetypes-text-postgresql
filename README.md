# @plurnk/plurnk-mimetypes-text-postgresql

text/x-postgresql-sql mimetype handler for plurnk-service. ANTLR-backed extraction using grammars-v4's `sql/postgresql` grammar.

## Symbols emitted

Per SPEC §3 inclusion policy: classes, functions, methods, fields, interfaces, enums, types, modules, variables, constants. Imports excluded; locals inside function bodies excluded.

## Install

```bash
npm install @plurnk/plurnk-mimetypes-text-postgresql
```

Auto-discovered by `@plurnk/plurnk-mimetypes` when installed alongside it.
